LINUX INSTALLATION OF MINECRAFT 1.8.9 AETHER
1. Ensure minecraft-launcher is installed
2. Install java 8. On Arch this can be done through pacman -S jre8-openjdk. Minecraft versions pre-1.13 don't work
on the latest version of java.
3. Create 1.8.9 install profile, save and launch it.
4. Download Forge 1.8.9 installer from http://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.8.9.html
5. Run java -jar ~/PATH_TO_INSTALLER
6. Install client
7. mv aether_legacy-*.jar ~/.minecraft/mods
8. launch minecraft and join server at 73.190.128.211
9. if you want optifine, download the jar and put it in mods.
