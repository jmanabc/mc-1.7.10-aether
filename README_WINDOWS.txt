WINDOWS INSTALLATION OF MINECRAFT 1.8.9 AETHER
1. Ensure minecraft is installed
2. Install java 8. On Windows this can be done from https://www.java.com/en/download/ good luck getting this to work
3. create 1.8.9 install profile, save and launch it.
4. Download Forge 1.8.9 installer from http://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.8.9.html
5. Double click the installer, install client.
6. put the included aether .jar into your mods folder
7. launch minecraft and join server at 73.190.128.211
8. If you want optifine, download the jar and put it in mods.
